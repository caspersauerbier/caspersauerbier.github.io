Swift / Objective-C

These languages are created by Apple for iOS and OS X development. Swift has been recently inroduced at Apple’s WWDC. Swift will eventualy replace Objective-C, because it is intended to be faster and safer than Objective-C. Both programming languages are designed to make applications using Xcode. HTML / CSS / JavaScript

These languages are created for web development. HTML is the standard programming language to create web pages. Web browsers can read HTML files and compose them into visible webpages. CSS is a style sheet language for describing the look and formatting of an HTLM file. It is used by most websites to create visually engaging webpages. JavaScript is a dynaming programming language, most commonly used as a part of a web browser.
